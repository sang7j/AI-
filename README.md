# AI 기반 감정·추상 키워드 도서 검색

감정/추상 키워드를 기반으로 책을 기록하고 검색하는 웹 서비스입니다.

## 배포 주소

GitHub Pages: https://sang7j.github.io/AI-/

## 주요 기능

- 네이버 책 검색 API 연동
- 감정/추상 키워드 등록 및 통계
- 키워드 동의/비동의 투표
- 로그인(이메일·비밀번호) 기능
- GitHub Pages + Supabase 배포
